* Item Swapper
  * It swaps Items using the Numpad
    * 4 - Item Slot 1
    * 5 - Item Slot 2
    * 6 - Item Slot 3
    * 1 - Item Slot 4
    * 2 - Item Slot 5
    * 3 - Item Slot 6
    * 0 - Reset
